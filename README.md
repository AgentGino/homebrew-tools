# AgentGino Homebrew Tools

Homebrew tap for AgentGino command-line tools.

## Install

```bash
brew tap AgentGino/tools
brew install stacky
```

You can also install a specific formula directly:

```bash
brew install AgentGino/tools/stacky
```

## Formulae

| Formula | Description |
|---|---|
| `gt` | AI-powered git commit tool that generates conventional commit messages. |
| `stacky` | CLI for deploying Supabase-like backends on your own AWS account. |

## Update

```bash
brew update
brew upgrade stacky
```

## Uninstall

```bash
brew uninstall stacky
brew untap AgentGino/tools
```
